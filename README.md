# 🏥 Patient Management System (Microservices)

Built a scalable **microservices-based healthcare backend** using Spring Boot, featuring API Gateway, authentication, event-driven processing, and cloud-ready infrastructure.

---

## 🚀 Tech Stack
- Java, Spring Boot, Spring Cloud Gateway  
- Spring Security, JWT  
- PostgreSQL  
- Kafka, gRPC (Protobuf)  
- Docker  
- LocalStack (AWS simulation)  
- JUnit, REST Assured  

---

## 🔑 Key Features
- Centralized **API Gateway** with routing and JWT validation  
- Secure **Auth Service** issuing and validating JWT tokens  
- **Patient Service** with full CRUD APIs  
- **gRPC communication** with Billing Service  
- **Kafka-based event streaming** to Analytics Service  
- **Dockerized microservices** with LocalStack for AWS services
